# @nowzoo/ngx-crumbs

Breadcrumbs and window title for Angular apps.

## Quick start

```bash
npm i @nowzoo/ngx-crumbs
```

Import the module...
```typescript
import { NgxCrumbsModule } from '@nowzoo/crumbs';
@NgModule({
  imports: [
    NgxCrumbsModule.forRoot()
  ],
})
export class AppModule { }
```
